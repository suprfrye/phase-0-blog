Ahh, enumerables, Ruby modules that help simplifiy tasks. Ruby is jammed pack with these special friends that make solving problems a little easier and solution code a more condensed. Today, we're going to look at Ruby's group_by method.

The group_by method is used to group collections of data by a common factor. The result is a hash with the common trait as the key and an array of the elements that share it as the value.
